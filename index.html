mailadmin by rek2

Postfix+Courier+Mysql Admin GUI in ruby and rails.

Mailadmin is an application with the objective to funtion as a front end of a Postfix+Courier+mysql mail server setup.. to be more expecific to this one: http://www.gentoo.org/doc/en/virt-mail-howto.xml But it can be changed easily to work with any other setup.

Dependencies

Postifx Courier-Imap MySql
